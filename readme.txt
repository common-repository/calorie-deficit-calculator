A WordPress plugin to calculate Calorie Deficit.

== Description ==

[Calorie Deficit Calculator](https://calculator-online.net/calorie-deficit-calculator/) is a free calculator plugin that enables you to estimate your weight loss and maintenance calories. Do not fret as the tool instantly tells you how much calorie deficit you need to achieve your weight goals.

== Features ==

1. Through this plugin users on your website can find following values: 

    * Weight Maintenance Calories
    * Weight Loss Calories
    * Calorie Deficit

2. It is very easy to use, You just have to use shortcode [cocdc-calorie-deficit] to show the calculator on webpage.

== Installation and Use ==

This section describes how to install and use this plugin on your dashboard.
* You need to upload the plugin files to the "wp-content\plugins" directory. Else, you can also install the plugin directly from Wordpress for free.
* Land on the Wordpress dashboard > Select Plugins > Activate Plugin.
* Add the tool on the front end by entering short code there [cocdc-calorie-deficit].
